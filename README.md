## 今回追加した機能
- 計算ホーム
- 17問のオリジナル計算問題

- 分野別正答率
- 問題ID表示

## 問題の品質チェック
正解・選択肢・途中式は同じ元データから生成しています。
起動時に次の項目を自動検査します。
- 選択肢が4つある
- 正解が1つだけある
- 選択肢が重複していない
- 途中式が存在する

誤答は、公式の使い方、百分率への変換、単位変換、掛け算と割り算など、実際に起こりやすいミスを想定しています。

公開後はURL末尾に `?v=31` を付けると古いキャッシュを避けやすいです。
